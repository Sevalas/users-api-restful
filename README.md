Esta es una aplicación desarrollada con:

- Java 17
- Spring Boot
- Banco de datos en memoria H2
- Proceso de build via Gradle
- Persistencia con Hibernate y JPA
- Servidor Tomcat
- Validaciones con JWT
- Swagger
